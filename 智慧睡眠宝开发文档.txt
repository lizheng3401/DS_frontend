1.注册
请求接口:http://sleeper.xiaopeng.site/api/Register
请求头:
Content-Type:application/json
请求方式:POST
数据(json格式):{"Phone":"18780271747","Password":"12345678","Name":"肖鹏","Sex":"男","BirthDate":"1994-03-02"}
格式化后：
{
    "Phone": "18780271747",
    "Password": "12345678",
    "Name": "肖鹏",
    "Sex": "男",
    "Birthdate": "1994-03-02"
}

服务器返回数据格式：(flag:0表示失败，1表示成功)
{
    "Flag": "0",
    "Message": "注册失败，您已注册！"
}
{
    "Flag": "1",
    "Message": "注册成功！"
}
